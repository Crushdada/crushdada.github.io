Vue常用指令详解

发表于 2021-02-17 更新于 2021-03-19 分类于 Code

vue指令

V-text、V-html、V-model

  1. v-text和v-html是单向绑定。

    1. {{}}插值表达式：将指定数据插入指定位置
    2. v-text='xx'，会覆盖掉元素原有内容
    3. v-html='xx'，也会覆盖掉原有内容，且会解析字符串中的标签元素(会转义)

  2. v-model为双向绑定，即view-model

    v-once

  3. 该指令只初始渲染一次，不会再变化。

  4. 不需要冒号表达式，如：<p v-once></p>

    v-cloak

  5. 不需要表达式

  6. 这个指令保持在元素上直到关联实例结束编译。

  7. 和 CSS 规则如[v-cloak] { display: none }一起用时，这个指令可以隐藏未编译的 Mustache 标签直到实例准备完毕。

v-if、v-else-if、v-else

条件渲染，指令取值为true则渲染，否则不会创建该元素

  1. 可以从模型中获取数据

  2. 也可以直接赋值一个表达式，如v-if='tmp > 10'

  3. v-if、v-else之间不能有其他元素

    v-show

  4. 也用于条件渲染

  5. 不同于v-if，它会创建元素

  6. 它设置了元素的display:none

  7. 适用于经常切换元素显示-隐藏状态时，避免元素多次创建、删除消耗性能

    v-for

  8. 根据数据多次渲染元素，通常用于列表渲染

  9. v-for 指令需要使用item in items形式的特殊语法 其中items是源数据数组，而item则是被迭代的数组元素的别名

  10. 或者通过(value,index)、(value,index)来循环有序的数组和无序的对象

  12. v-for=”(value,key) in obj”，循环对象时，因为对象无序，所以并非是(各项,索引)的形式，而是(值,名)。vue会将对象中各个键值对的值赋给第一个变量，名赋给第二个变量(与左右顺序有关，变量名无关)

  13. v-for=”(item,index) in items”，循环对象数组时，数组有序，符合(各项,索引)的形式 ，其中每一项item为一个对象，因此可用对象.属性名调用。
